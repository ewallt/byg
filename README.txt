 # Behold Your God - Chapter Summaries



This is a monorepo containing a collection of self-contained, single-page web applications. Each application is an interactive summary of a chapter from the "Behold Your God" text. A combined app with multiple chapters is envisioned.



## Structure



Each folder in the root of this repository is a separate, independent web app, plus knowledge base.



- `/chapter1`

- `/chapter2`

- etc.



## Usage



To view an app locally, open the `index.html` file from within its respective chapter folder in a web browser.
